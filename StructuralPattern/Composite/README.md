# Composite Pattern

## 案例需求-设计杀毒软件的树形框架结构
<p>Sunny软件公司欲开发一个杀毒(AntiVirus)软件，该软件既可以对某个文件夹(Folder)杀毒
，也可以对某个指定的文件(File)进行杀毒。该杀毒软件还可以根据各类文件的特点，为不同类型的
文件提供不同的杀毒方式，例如图像文件(ImageFile)和文本文件(TextFile)的杀毒方式就有所差
异。现需要提供该杀毒软件的整体框架设计方案。</p>
